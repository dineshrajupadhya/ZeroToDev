## An Array is a multi-valued datatype with ordered collection of data items having same name and index values, separated with commas. The Array in Python can be enclosed between list, set, tuple and dictionaries.
## Indedxing is possible in array in python.
## Declaration of Array:
## Syntax: List Declaration:
## Array_name = [values separated by commas]
## Tuple declaration:
## Array_name = (values seperaated by commas)
## Set declaration:
## Array_name = {values separated by commas, the values are unordered}
## Dictionary declation:
## Array_name = {Key1:value1, key2:value2,….has key-value pairs separated by comma(,)}
